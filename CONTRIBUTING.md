# Contributing

Thanks for your interest in SwiftDiffusion!

## How to Contribute

1. **Fork** the repository.
2. Create a new branch: `git checkout -b feature/my-feature`.
3. Make your changes.
4. Test on your GPU (GTX 1060 or similar recommended).
5. Commit and push to your fork.
6. Open a **Pull Request** using the template.

## Guidelines

- Keep the codebase minimal – no unnecessary dependencies.
- Follow the existing code style (imports, naming, formatting).
- If you add UI text, update both `locales/en.json` and `locales/pl.json`.
- Test that the application starts and runs without errors.

## Development Setup

Run `install.bat` to set up the virtual environment, then `start.bat` to launch.

For questions, use the [Discussions](https://github.com/AnonBOTpl/SwiftDiffusion/discussions) tab.
